Juego DonkeyKong Version Nintendon't

Para instalar el juego solo necesita ejecutarse el archivo.jar.

Se debe tener un sistema operativo Windows Xp o superior.

Se debe tener un JDK java 8 o superior.


Instrucciones del juego:

Para comenzar, se debe presionar la tecla 'y', para pasar a la seleccion de personaje.

Los personajes disponibles en esta version son: Mario y Pikachu.

Presionar 'y' para comenzar el juego.

Teclas comando:

'a': avanza personaje hacia la izquierda.
's': baja las escaleras, una vez que sube no puede bajar a un piso inferior.
'd': avanza personaje hacia la derecha.
'w': saltar, subir escaleras.
'f': lanza poder especial.
'p': pausa.
'n': vuelve a pagina principal.



Cada vez que se supera un nivel, se suma al score total, una vez que este supere el highscore anterior, 
se regista este ultimo, apareciendo en la pantalla principal.

Para obtener POWER-UPS: se debe recoger una estrella que aprece aleatoriamente en el mapa.
